# Support

`othellopy` is maintained as an open source educational Python package.
The official distribution is the PyPI package
[`othellopy`](https://pypi.org/project/othellopy/) built from
[`Hietan/othellopy`](https://github.com/Hietan/othellopy).

Use GitHub issues for reproducible bugs and scoped feature requests. For usage
questions, include a minimal code sample, the Python version, and the installed
`othellopy` version or commit.

For PyPI installations, include the output of:

```bash
python -m pip show othellopy
```

Installs from GitHub branches are useful for development and testing, but they
are not the primary support target.

Maintainers may close issues that do not include enough information to
reproduce, are unrelated to this package, or request private tutoring or
custom implementation work.
