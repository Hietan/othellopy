# Maintainers

`othellopy` is currently maintained by Hietan.

Maintainers are responsible for:

- Reviewing pull requests and enforcing the GitFlow branch model.
- Keeping the PyPI package
  [`othellopy`](https://pypi.org/project/othellopy/) aligned with this
  repository.
- Managing release tags, the `pypi` GitHub Environment, and PyPI Trusted
  Publishing.
- Reviewing Dependabot pull requests before they enter `dev`.
- Handling security reports privately until disclosure is appropriate.

New maintainers should be added only after sustained, high-quality
contributions and explicit approval from the existing maintainer.
